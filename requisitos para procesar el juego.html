Fortnite es un juego el cual se puede encontrar y jugar de forma gratuita pero al ser gratis también se necesita unos requisitos para poder jugarlo sin problemas en un ordenador.

La memoria RAM del ordenador

La memoria ram es la que le da la fuidez a un ordenador y le da rapidez para el procesado de tareas sin tener un inconveniente o interrupción al dessarrollar estas, en pocas palabras, dependiendo de la RAM del ordenador, asi de rápido será el ordenador.
El juego para jugarse necesita como mínimo 8GB de memoria RAM para llevarse a cabo, de lo contrario, tendrá problemas con fluidez y será muy dificil jugarlo.

Windows 7 - 8 - 10 de 64 bits

El procesador de tu ordenador debe ser cualquiera de estos pero de 64 bits para poder llevarlo a cabo, no importa la versión, ya sea pro, home o similares.
Esto se puede conocer al comprar el ordenador o simplemente llendo al Panel de control, Sistemas y especificaciones de este

Gb de VRAM

La VRAM es lo que se conoce como la Tarjeta Gráfica es un tipo de memoria RAM que utiliza el controlador gráfico para poder manejar toda la información visual que le envia la CPU del sistema. La principal característica de esta clase de memoria es que es accesible de forma simultánea por dos dispositivos. En teoría es la capacidad de procesado de imagen que envía el CPU al monitor. Para correr el juego se necesitan 2GB de VRAM en el ordenador para llevarse a cabo.

Nvidia GTX 660 o AMD Radeon HD 7870 con GPU DX11 equivalente.

Nvidia o ADM son procesadores gráficos que tienen los ordenadores, todos los ordenadores poseen uno de estos.
Para correr Fortnite se necesita o una gráfica Nvidia GTX 660 o AMD Radeon HD 7870 con GPU DX11 equivalente.
>

Core y GHz
Core i5 es una nomenclatura que designa procesadores de gama media o media-alta de la marca Intel. Se caracterizan por un precio asequible y prestaciones decentes. Suelen emplearse en ordenadores preparados para ejecutar programas complejos o juegos que necesiten potencia ligeramente superior. La familia i5 ofrece una velocidad de procesamiento media de unos 3.5 GHz y un caché de unos 8 Mb.
